# 更新策略 Security Policy

## 支持的版本 Supported Versions

| Version | 状态 | 说明                                        |
| ------- | ---- | ------------------------------------------- |
| 1.x     | 🚧   | 仅接受安全漏洞修复和关键Bug修复，不新增功能 |
| 2.x     | ✅   | 持续接收新功能、改进和安全修复              |

## 漏洞报告 Reporting a Vulnerability

如果你发现了安全漏洞，请提 issue。
